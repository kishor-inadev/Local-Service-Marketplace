The gateway will act as the single entry point for all client requests.
 
---
 
Technology stack
 
NestJS
Node.js
Axios for service communication
Redis (optional caching / rate limiting)
Docker
 
---
 
Architecture rules
 
- The gateway must not contain business logic
- The gateway only routes requests to microservices
- Validate JWT tokens before forwarding requests
- Implement request logging
- Implement rate limiting
- Handle CORS
- Support centralized error handling
 
---

Route incoming requests to the appropriate microservice.
